# Spatial Skills Game

A small desktop Unity project for spatial reasoning practice. The game asks the player to rotate a 3D block shape until it matches the target orientation.

## Open the project

1. Open this folder in Unity Hub.
2. Use Unity 2022.3 LTS or newer.
3. Open `Assets/Scenes/Main.unity`.
4. Press Play.

## Gameplay

- Drag the model to rotate it.
- Use the buttons to reset, snap to 90-degree angles, check the answer, and advance.
- The project is designed for desktop builds and uses Unity's built-in render pipeline and uGUI.

## Build target

Use `File > Build Settings`, choose `PC, Mac & Linux Standalone`, then build for your desktop platform.
